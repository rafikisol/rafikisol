### Rafikisol ###
## What is Rafikisol?##
Rafikisol (solemn friend) is meant to combine Google Earth Engine (GEE), rgee, tidymodels and more into models to use
for every one who might need them. We are open to new ideas and critisims and all the papers that the code is translated 
to is cited (please uses theirs where needed). However, our own ideas are free to do with what you may. We believen 
open sources and information is free not a luxury, therefore, if the code does not provide enough information or something
is wrong please contact us or if you want to collaborate, that would be great.
